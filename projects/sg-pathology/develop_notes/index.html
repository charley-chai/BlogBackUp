SwanGeese Pathology 的开发笔记

OCT 01, 2018 by USTC SwanGeese Team

之所以要写只得东西，一方面是为了防止遗忘。更重要的是我们希望可以帮助更多跟我们遇到一样问题但至今没有解决的人。

基本步骤

#001

下载OpenSlide for windows


前往OpenSlide官网下载OpenSlide的Windows Binaries. 选择合适的操作系统位数。




#002

提取需要的dll文件


由于 OpenSlideNET 代码本身是基于 OpenSlide for C，因此需要将 OpenSlide for C 中的 libopenslide-0.dll 及其依赖 库手动添加至C#开发环境中。

即将解压后文件夹中的bin目录下的dll全部拷贝至开发的C# .NET项目的/bin/Debug目录。 (该图片文件不全，需拷贝目录下所有dll文件)




#003

安装OpenSlideNET


前往OpenSlideNET仓库下载OpenSlideNET源码并解压。


#004

添加OpenSlideNET引用


将OpenSlideNET项目下的src -> OpenSlideNET -> OpenSlideNET.csproj 通过添加现有项目添加至正在开发的 C#项目中。并在正在开发的C#项目中添加引用 -> 项目 -> 勾选OpenSlideNET -> 点击确定添加。




#005

配置Targetframework


右键点击OpenSlideNET项目进入属性配置，在生成标签页内，将目标平台设置为x64或x86(与下载的安装包一 致)，对开发的项目做相同的操作。




#006

构建整个工程


在开发项目的引用部分输入 using OpenSlideNET，而后即可使用。


#007

清理项目


在运行一次过后，OpenSlideNET的dll以及它的依赖包已被加载进开发项目的/bin/Debug目录下。 因此可以在开发项目的引用中移除OpenSlideNET，并在解决方案中移除OpenSlideNET项目。 最后在开发项目中引用自身项目目录中的/bin/Debug/OpenSlideNET.dll.
