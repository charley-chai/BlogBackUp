在C#中调用Python

OCT 2, 2018 by Charley Chai

本文介绍如何在Visual Studio 2017开发环境下，在WPF应用中使用C#来调用Python的配置方法。

基本步骤

#001

安装Python


首先，在Visual Studio 2017中安装 Python tools for Visual Studio。

打开Visual Studio Installer，在 Workloads 项下面勾选 Python development 或者 Data Science and analytical application。




#002

创建WPF项目


就仅仅是正常创建而已，不再赘述。


#003

安装IronPython


使用Nuget搜索 IronPython 安装即可。




测试

IronPython 允许我们直接运行字符串的形式 Python 程序，也可以将引用一个 Python 文件，这里采用后者进行测试。

在任一位置新建一个 test.py 文件，内容如下：

                        
def SayHello():
    return "Hello world"

def Add(a, b):
    return a + b

而对应的 C# 代码为：

                        
using IronPython.Hosting;

namespace YourApp
{

    public partial class MainWindow:Window
    {

        public MainWindow()
        {
            InitializeComponent();
        }

        private test()
        {
            var engine = Python.CreateEngine();
            var scope = engine.CreateScope();
            var source = engine.CreateScriptSourceFromFile("hello.py");
            
            source.Execute(scope);
            var sayHello = scope.GetVariable>("SayHello");
    
            MessageBox.show(sayHello().ToString()); // Hello World 
    
            
            var add = scope.GetVariable>("add");
            var result1 = add(1, 2);
            MessageBox.show(result1.ToString()); // 3

            var result2 = add("hello ", "world");
            MessageBox.show(result2.ToString()); // Hello World

        }
    }

}
