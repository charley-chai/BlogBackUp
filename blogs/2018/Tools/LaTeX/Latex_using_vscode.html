在Visual Studio Code中使用LaTeX

SEP 21, 2018

LATEX简介

下面是一段来自维基百科的文字：

LATEX（文本写作LaTex，后文采用此种写法。）是一个基于TEX的文字排版系统。它 由美国计算机科学家莱斯利·兰伯特在20世纪80年代初期开发，利用这种格式系统的处理，即使用户没有排版和程序设计的知识也可以充分发挥由TeX所提供的强大功能， 不必一一亲自去设计或校对，能在几天，甚至几小时内生成很多具有书籍质量的印刷品。对于生成复杂表格和数学公式，这一点表现得尤为突出。 因此它非常适用于生成高印刷质量的科技和数学、化学文档。这个系统同样适用于生成从简单的信件到完整书籍的所有其他种类的文档。

简而言之，LaTeX就是一个帮助我们排版的工具。我们使用LaTeX语法快捷地编写我们想要呈现的内容，然后我们得到由LaTeX生成的精美排版的PDF文件。无论是复杂的数学公式，还是图像， 各国语言它都可以胜任。

这里又有提到，LaTeX是基于TeX开发的，有必要介绍一下TeX。TeX是由Donald E. Knuth开发的功能强大的幕后排版系统，它是Knuth在写他 的大作TAOCP(The Art Of Computing Science)时，由于对出版商糟糕的排版十分不满，所以他开发出了这套系统。稍后还会写一些东西来 介绍一下这位神人。

Donald E. Knuth Leslie Lamport

使用什么编写LATEX

有很多软件可供我们使用LATEX，维基百科上有一个很全面的对比可以参考。下面的列出一些：

  • LyX: 所见即所得，支持Windows，macOS，Linux，OS/2等。

  • TeXstudio: 编译和预览同时可见，支持Windows，macOS，Linux，OS/2等。

  • Sublime Text：代码高亮效果好，颜值高，插件丰富，文件管理方便但是“付费”。要实现编辑、预览、编译生成一体化，需要配置一下。支持Windows，macOS，Linux。

  • Visual Studio Code：当然，今天我要介绍的还是使用Visual Studio Code来实现将编辑，预览及生成集为一体。首先，简单介绍一下下本人目前的主力编辑器Visual Studio Code，它由微软开发，由Typescript语言。我个人觉得其在各个方面都可媲美Sublime Text，而且有些方面还要更强一些(啊啊啊。。。为巨硬疯狂打call！！！)。

在macOS下使用Visual Studio Code编辑LATEX

准备工作

首先我们需要在Mac上安装Visual Studio Code（以下简称VS Code）和MacTeX（你也可以选择其他发行版，不过并不保证后面的做法有效）。这里的 就是普通下载安装就行了。

安装VS Code插件

接下来我们需要做的是安装VS Code上的LaTeX插件。

其中LaTeX Workshop是必须要装的。此外推荐LaTeX language support插件，它提供更好的语法高亮等功能。

安装插件的方法也很简单，此处不再赘述。

设置VS Code

这里我们做两件事

  • 在latex-workshop.latex.tools配置中，增加xelatex项。

  • 在latex-workshop.latex.recipes配置中，将第一项的latexmk改为xelatex。

至于原因，由于本人对此插件的了解程度不是很深，就。。。就不班门弄斧了（😁）

那么应该怎么具体操作呢？很简单：

  • 如何找到设置

    在菜单栏中Code > Preferences > Settings打开设置界面

  • 修改用户设置

    在上面搜索栏中对latex-workshop.latex.tools进行搜索定位

    得到上图所示的界面之后，在右侧的用户设置区将下面的代码复制进行就OK了。

                                    
{
    "latex-workshop.latex.tools": [
        {
            "name": "xelatex",
            "command": "xelatex",
            "args": [
                "-synctex=1",
                "-interaction=nonstopmode",
                "-file-line-error",
                "-pdf",
                "%DOC%"
            ]
        },
        {
            "name": "latexmk",
            "command": "latexmk",
            "args": [
                "-synctex=1",
                "-interaction=nonstopmode",
                "-file-line-error",
                "-pdf",
                "%DOC%"
            ]
        },
        {
            "name": "pdflatex",
            "command": "pdflatex",
            "args": [
                "-synctex=1",
                "-interaction=nonstopmode",
                "-file-line-error",
                "%DOC%"
            ]
        },
        {
            "name": "bibtex",
            "command": "bibtex",
            "args": [
                "%DOCFILE%"
            ]
        }
    ],

    "latex-workshop.latex.recipes": [
        {
            "name": "xelatex",
            "tools": [
                "xelatex"
            ]
        },
        {
            "name": "pdflatex -> bibtex -> pdflatex*2",
            "tools": [
                "pdflatex",
                "bibtex",
                "pdflatex",
                "pdflatex"
            ]
        }
    ]
}

测试LATEX效果

在设置好之后，重启VS Code。下面我们就开始测试我们的工作是否有效。

首先，在某个目录线新建一个LaTeX文件，这里我们不妨叫它helloworld.tex

再次文件中输入一下LaTeX代码：

                        
\documentclass[UTF8]{ctexart}
\usepackage{amsmath}
\usepackage{xfrac}
\begin{document}
Hello world!

你好世界！
\end{document}

当我们保存之后(CMD + S)，VS Code就会在本目录下生成一堆文件，其中有一个helloworld.pdf 的文件，那正是我们需要的。同时VS Code还支持生成PDF文件的“预览”，只需点击界面右上角的一个有放大镜的图标即可。 效果如图所示：
